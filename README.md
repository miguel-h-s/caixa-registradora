# Caixa Registradora em Go

## O que o programa faz
- Pede o valor de um produto pelo terminal
- Valida o valor informado
- Atualiza o total acumulado das compras
- Permite visualizar o total a qualquer momento

## O que aprendi / reforcei
- Funções
- Loops
- Variáveis
- Estruturas condicionais (`if / else`)
- Organização básica de um programa interativo

## Exemplo de uso
1. O cliente chega ao caixa registradora com um produto  
   (exemplo: um pacote de arroz no valor de R$ 26,00)
2. O operador seleciona a opção **“registrar valor de compra”**
3. Digita o valor do produto
4. Após registrar todos os itens, seleciona a opção  
   **“mostrar total acumulado”** para ver o valor final da compra

## Nota
Este projeto foi desenvolvido com foco em aprendizado e prática de lógica de programação,
não sendo um sistema real de caixa registradora.
